# ShortStay - Online Marketplace for Short-Term Accommodations

A full-stack web application for booking short-term accommodations with role-based access control.

## Features

- **User Roles**: Guest, Host, Admin, Payment Manager, Field Inspector
- **Property Management**: List, search, and book properties
- **Booking System**: Complete booking workflow with payment tracking
- **Reviews & Ratings**: User review system for properties
- **Admin Dashboard**: Full administrative control panel
- **Responsive Design**: Mobile-first approach with Tailwind CSS

## Tech Stack

### Backend
- Node.js + Express.js
- MySQL with Sequelize ORM
- JWT Authentication
- Multer for file uploads

### Frontend
- React.js with Hooks
- React Router v6
- Tailwind CSS for styling
- Axios for API calls
- Chart.js for analytics
